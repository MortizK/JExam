# JExam

A local, open-source exam generator.

## Features (v1)

The ones currently Implemented + All XML Handling + Autosave + PDF Generation

## Features (v2)

Multiple Language Support (Mainly English and German)

Support for Images and Formular in the Questions

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/MortizK/JExam.git
```

2. Build with Maven

```bash
./mvnw clean install
```

Common Error while Compiling: A Process is still using the ``\target``-Folder

3. Run the application

```bash
java -jar .\target\JExam-1.0-SNAPSHOT.jar
```
